# Multi-Label Classification of Chest X-Ray Images Using CNNs

## Overview
This project implements a Convolutional Neural Network (CNN) for multi-label classification of chest X-ray images. The model predicts the presence of multiple thoracic diseases from a single X-ray image, demonstrating the application of deep learning in medical image analysis.

## Features
- Multi-label classification of chest X-ray images
- Image preprocessing and data visualization
- CNN-based deep learning model
- Model training and performance evaluation
- Built using PyTorch

## Tech Stack
- Python
- PyTorch
- NumPy
- Pandas
- Matplotlib
- OpenCV

## Dataset

The dataset used in this project is available on Kaggle:

**NIH Chest X-ray Dataset:**  
https://www.kaggle.com/datasets/nih-chest-xrays/data

## Results
The model was trained to identify multiple disease labels from chest X-ray images and evaluated using standard classification metrics.

## Project Status

This is an ongoing project. Current efforts are focused on improving model performance through experimentation with different CNN architectures, hyperparameter tuning, and enhanced evaluation metrics to increase prediction accuracy and reliability.
